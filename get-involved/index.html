Making Changes

If you've got free time, feel free to add a new class on GitHub or refine some code that could be more user-friendly or efficient!

Making a Plugin

If you've created a plugin for dars and want it usable with the package manager (packman) you can fork the repository, add it to packman.py and pull request it.


Have fun!
